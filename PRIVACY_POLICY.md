# Privacy Policy / プライバシーポリシー

Last updated: 2026-01-02

## English

### Data Collection

Backlog Quick collects and stores the following data locally on your device:

- **Backlog API Key**: Used to authenticate with your Backlog account
- **Backlog Space URL**: Your Backlog workspace URL (e.g., yourspace.backlog.com)
- **Default Settings**: Your preferred project, issue type, priority, and assignee settings
- **Screenshots**: Temporarily captured for issue attachment (deleted after use)

### Data Usage

- All data is used solely to interact with the Backlog API on your behalf
- Screenshots are captured only when you explicitly click the screenshot button
- Temporary data is automatically deleted after issue creation

### Data Storage

- All data is stored locally in Chrome's extension storage
- No data is sent to third-party servers
- Data is transmitted only to your configured Backlog space via HTTPS

### Data Sharing

- We do not collect, share, or sell any user data to third parties
- Your API key and data are only transmitted to your own Backlog instance

### Security

- All communication with Backlog uses HTTPS encryption
- API keys are stored in Chrome's secure local storage
- No analytics or tracking is implemented

### Your Rights

- You can delete all stored data by removing the extension
- You can view stored data in Chrome's extension settings

### Contact

For questions about this privacy policy, please open an issue at:
https://github.com/d8y/backlog-quick/issues

## 日本語

### 収集するデータ

Backlog Quickは以下のデータをお使いのデバイスにローカル保存します：

- **Backlog APIキー**: Backlogアカウントの認証に使用
- **BacklogスペースURL**: お使いのBacklogワークスペースURL（例: yourspace.backlog.com）
- **デフォルト設定**: プロジェクト、課題タイプ、優先度、担当者の設定
- **スクリーンショット**: 課題添付用に一時的にキャプチャ（使用後に削除）

### データの利用

- すべてのデータはBacklog APIとの通信にのみ使用されます
- スクリーンショットは明示的にボタンをクリックした場合のみキャプチャされます
- 一時データは課題作成後に自動削除されます

### データの保存

- すべてのデータはChromeの拡張機能ストレージにローカル保存されます
- サードパーティサーバーへのデータ送信はありません
- データは設定されたBacklogスペースにHTTPSでのみ送信されます

### データの共有

- ユーザーデータを第三者に収集・共有・販売することはありません
- APIキーとデータはお客様自身のBacklogインスタンスにのみ送信されます

### セキュリティ

- Backlogとの通信はすべてHTTPS暗号化を使用
- APIキーはChromeの安全なローカルストレージに保存
- アナリティクスやトラッキングは実装していません

### ユーザーの権利

- 拡張機能を削除することで、保存されたすべてのデータを削除できます
- Chromeの拡張機能設定で保存データを確認できます

### お問い合わせ

このプライバシーポリシーに関するご質問は、以下でIssueを作成してください：
https://github.com/d8y/backlog-quick/issues
